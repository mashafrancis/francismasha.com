**JavaScript version**

```js
const plusOneYear = ((d) => new Date(d.setFullYear(d.getFullYear() + 1)))(
	new Date(),
);
```

**TypeScript version**

```js
const plusOneYear: Date = ((d) => new Date(d.setFullYear(d.getFullYear() + 1)))(
	new Date(),
);
```
